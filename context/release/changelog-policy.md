# Changelog policy (Keep a Changelog + SemVer)

- `CHANGELOG.md` siempre tiene `## [Unreleased]`.
- PR con cambio notable -> añadir entrada en Unreleased:
  - Added / Changed / Deprecated / Removed / Fixed / Security

Versionado:
- SemVer: MAJOR.MINOR.PATCH
- Release: `## [X.Y.Z] - YYYY-MM-DD`

Regla: texto “para humanos”, no logs automáticos.
